プロジェクト全体の品質チェックを実行してください。

## 手順

### 1. Lint チェック
```bash
pnpm lint
```
エラーがあれば `pnpm lint:fix` で自動修正。

### 2. 型チェック
```bash
npx tsc --noEmit
```
型エラーがあれば修正。

### 3. ユニットテスト
```bash
pnpm test
```
失敗したテストがあれば原因を分析し修正。

### 4. ビルド確認
```bash
pnpm build
```
ビルドエラーがあれば修正。

## 結果報告

各ステップの結果をまとめて報告:
- ✓ Lint: パス/エラー数
- ✓ 型チェック: パス/エラー数
- ✓ テスト: パス/失敗数
- ✓ ビルド: 成功/失敗
