プロジェクトをビルドし、エラーがないことを確認してください。

## 手順

1. `pnpm build` を実行
2. ビルドエラーがあれば原因を分析
3. エラーを修正して再ビルド
4. 成功するまで繰り返す

## よくあるエラー

- **型エラー**: TypeScript の型不整合を修正
- **インポートエラー**: パスやモジュール名を確認 (`@/*` → `./src/*`)
- **未使用変数/インポート**: `pnpm lint:fix` で自動修正
- **ページの default export 不足**: Next.js ページは default export が必要
